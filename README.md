# 🚀 Lead Management System - Mini CRM

A modern full-stack Lead Management System built using **React**, **Node.js**, **Express.js**, and **PostgreSQL**.

This project allows users to manage leads efficiently with features like lead creation, status updates, filtering, searching, and dashboard analytics.

---

# 📌 Features

## ✅ Lead Management
- Add new leads
- View all leads
- Update lead status
- Delete leads

## ✅ Lead Status
- Interested
- Not Interested
- Converted

## ✅ Search & Filters
- Search by:
  - Name
  - Phone
  - Source
  - Status
- Filter by source
- Filter by status

## ✅ Dashboard Analytics
- Total leads
- Interested leads
- Converted leads
- Not interested leads
- Conversion rate

## ✅ Validation
- Name validation
- Phone number validation

## ✅ UI Features
- Responsive layout
- Modern CRM dashboard UI
- Clean table design
- Sidebar navigation
- Hero section
- Analytics cards
